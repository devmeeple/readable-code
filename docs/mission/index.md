# 스터디 카페 이용권 선택 시스템

> 리팩터링을 왜 진행했는지, 의도를 밝히시오.

## 🚀 기능 요구사항

- 3가지 방법 중 1가지를 선택할 수 있다.
    - 시간권: 2, 4, 6, 8, 10, 12시간
    - 주권: 1, 2, 3, 4, 12주
    - 고정석: 4주, 12주
- 고정석은 추가 금액을 지불하면 사물함을 이용할 수 있다.
- 선택한 옵션의 최종 금액을 제공한다. (이용권, 사물함)
- 현재 오픈 이벤트로 할인 중이다.
    - 2주권 이상: 10%
    - 12주권:  15%

## ✔️ 리팩터링 포인트

- 추상화 레벨
- OOP
    - 책임과 협력(SRP, DIP)
    - 일급 컬렉션
